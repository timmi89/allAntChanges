Get the “React Button”

React ← Drag this link to your Bookmarks Bar

To install the React button in Firefox:

  1. Display your Bookmarks bar by clicking View > Toolbars > Bookmarks Toolbar
  2. Drag the “React” button to your Bookmarks bar
  3. When you are browsing the web, push the “React” button to react to anything on the page.

To install the React button in Chrome:

  1. Display your Bookmarks bar by clicking the Wrench Icon > Tools > Always Show Bookmarks Bar
  2. Drag the “React” button to your Bookmarks bar
  3. When you are browsing the web, push the “React” button to react to anything on the page.

To install the React button in Safari:

  1. Display your Bookmarks bar by clicking View > Show Bookmarks Bar
  2. Drag the “React” button to your Bookmarks bar
  3. When you are browsing the web, push the “React” button to react to anything on the page.

To install the React button in Internet Explorer:

  1. Display your Favorites Bar by clicking Tools > Toolbars > Favorites Bar
  2. Right-click the “React” button and select “Add to Favorites”
  3. On the pop-up window, select “Create in: Favorites Bar”
  4. When you are browsing the web, push the “React” button to react to anything on the page.
